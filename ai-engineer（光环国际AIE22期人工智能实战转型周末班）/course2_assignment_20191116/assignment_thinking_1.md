1，硬币问题：
P(正|A) = 0.7
P(正|B) = 0.4
P(A) = P(B) = 0.5
P(正) = 0.55

拿一次 为正的 请问是硬币A还是B？
P(A|正) = ？ P(正|A)*P(A) / P(正) = 0.7 * 0.5 /0.55 = 7/11
P(B|正) = ？ P(正|B)*P(B) / P(正) = 0.4 * 0.5 /0.55 = 4/11

是A的概率为 7/11
是B的概率为 4/11

拿10次 6次为正 请问是硬币A还是B？
P(正|A) = C(6,10)* 0.7**6 * 0.3**4
P(正|B) = C(6,10)* 0.4**6 * 0.6**4

P(A|正) = P(A) * P(正|A) /P(正) 
P(B|正) = P(B) * P(正|A) /P(正)

P(A) = P(B)

? P(A|正)/ P(B|正) 
简化后得：
P(A|正)/ P(B|正)  = P(正|A) / P(正|B) = (0.7**6 * 0.3**4) / (0.4**6 * 0.6 **4) 
=  1.7951812744140612 > 1

最后得出 是A的概率大。
